# Bempp-UQ
A Python package on top of [Bempp 3.3.4](https://bempp.com/bempp334/) to perform shape UQ.

## Installation
```
python -m venv .venv
source .venv/bin/activate

pip install -r requirements.txt
pip install -e .
```

## pre-commit
```
pre-commit install
```
